# Never Break Rules

Every rule in `.claude/rules/` is active at all times. Never skip, shortcut, or override any rule unless {{PRINCIPAL}} explicitly says to bypass it in that message.

**The standard:** If a rule applies to what you're about to do, follow it — even if it means pausing to ask first. "I forgot" or "I thought it was implied" are not valid reasons to skip a rule.

If {{PRINCIPAL}} explicitly says "skip X rule" / "bypass that" in the current message, proceed without it. Authorization must be explicit and in that message.
