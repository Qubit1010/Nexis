# Skill Creation

When {{PRINCIPAL}} asks to create/build a new skill:
- **They name the skill-creator** → proceed with the `skill-creator` workflow.
- **They don't** → ask first: "Build this with the `skill-creator` workflow (SKILL.md + evals), or just scaffold it directly?" Wait, then proceed.

Default recommendation: the `skill-creator` workflow. Build skills only when a workflow actually repeats.
