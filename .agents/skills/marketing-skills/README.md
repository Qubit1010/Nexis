## Skill-by-skill

### Traffic & Discovery

| Skill | What it does | When to invoke |
|-------|-------------|----------------|
| `seo-audit` | Full technical & on-page SEO audit with fixes | "audit my site seo", "why am i not ranking" |
| `ai-seo` | Optimize content for LLMs / AI search (ChatGPT, Perplexity, Gemini) | "optimize for ai search", "get cited by chatgpt" |
| `programmatic-seo` | Build SEO program — landing pages at scale, templates, automation | "build programmatic seo pages", "scale my seo" |
| `site-architecture` | Structure URLs, navigation, internal linking for SEO | "fix my site structure", "improve internal links" |

## Common combos

| Goal | Skills to chain |
|------|----------------|
| Win a new client | `competitor-profiling` → `cold-email` or `prospecting` → `sales-enablement` |
| Launch a product | `offers` → `pricing` → `copywriting` + `image` + `video` → `launch` + `co-marketing` + `public-relations` + `referrals` |
| Fix a leaky funnel | `analytics` → `cro` + `signup` + `popups` → `onboarding` → `churn-prevention` |
| Grow organic traffic | `seo-audit` → `site-architecture` + `schema` → `content-strategy` → `programmatic-seo` |
| Build a content engine | `customer-research` → `marketing-psychology` → `content-strategy` → `copywriting` → `social` + `emails` |
| Run paid media | `ad-creative` → `ads` → `analytics` → `ab-testing` |
| Monetize existing users | `pricing` → `paywalls` → `offers` → `emails` → `referrals` |
