## 🔑 Step 3: Messaging Funnel — 3-Touch DM Flow

### ✍️ DM 1 — Welcome + Qualifying

WHEN: Send when the member joins or if they’ve never been messaged beforeGOAL: Warm connection + qualify with one simple question

Framework:

```
Hey {firstname}, welcome to the [insert group name] group!

This group is for [insert niche: "agency founders"] looking to [insert transformation: "book more qualified sales calls using AI + multi-channel outbound systems"] without [insert pain: "hiring setters" / "relying on referrals" / "doing manual outreach"].

Just curious — are you currently [insert context-based question: "doing outreach manually or using any automation yet?"]

```

### ✍️ DM 1 (Alternate) — For Old/Existing Members

WHEN: Use for members already in the group, but never messagedGOAL: Same as above — qualify + open convo

Framework:

```
Hey {firstname}, noticed you’ve been part of the [insert group name] group for a while now.

This group is for [insert niche] looking to [insert transformation or end goal] without [insert pain].

I’m curious — are you currently [insert qualifying question]?

```

### ✍️ DM 2 — Value Giveaway

WHEN: Send 5 days after DM 1 if no response or cold responseGOAL: Deliver value (AI resource library, training, etc.) and reopen the loop

Framework:

```
Hey {firstname},  

We’re giving a few [insert niche: "agency owners"] early access to our [insert value: "library of AI Outbound Guides / mini-course / SOP vault"] to help them [insert outcome: "build a predictable appointment-setting system using AI, cold email + LinkedIn"].

Want me to send it over?

```

### ✍️ DM 3 — Re-engagement + Audit/Call

WHEN: 2 days after DM 2GOAL: Final touch — offer a free audit or invite for a walkthrough call

Framework:

```
Hey {firstname}, just circling back to see if you got a chance to check my last message.

Also, happy to share a [insert offer: "free audit of your outbound strategy"] or walk you through how we help [insert ICP: "agencies"] [insert result: "book 10–15+ qualified sales calls/month using AI + LinkedIn + cold email"].

Would you be open to that?

```

## 📌 Bonus DM — For Engaged Members (Commented or Liked Posts)

WHEN: Send when they interact with group contentGOAL: Start conversation while they’re warm

Framework:

```
Hey {firstname}, saw your comment on [insert post topic or keyword].

Quick Q — are you currently [insert qualifying question: "doing outreach manually or have something automated in place?"]

We help [insert ICP] build [insert transformation: "a scalable system using AI + outbound to book 10–15 calls/month"].

Want me to break down how this could work for you?

```
