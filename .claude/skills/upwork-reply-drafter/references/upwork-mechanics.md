# Upwork Platform Mechanics — the hard rules a reply must respect

Cited cheat-sheet distilled from `research-synthesis.md` (Q3-Q7). These are the platform facts a drafted reply must never contradict. When a reply implies an action, it must be the correct Upwork action.

## JSS (Job Success Score) — protect it in every message
- Rolling metric, recalculated **daily**, over **6 / 12 / 24-month** windows (Q4).
- Weighs public stars + **private feedback (invisible, highest weight)** + contract outcomes + earnings + relationship length (Q4).
- **Never advise closing a contract yourself** — freelancer-initiated closes are a negative signal. The client should close it (Q4/Q5).
- **A sour project → mutual close or partial refund**, never a formal dispute if avoidable (Q4).
- **Document scope + resolutions in the message thread** — the paper trail prevents the disputes that tank the score (Q4/Q7).

## Contracts, milestones, escrow (fixed-price)
- Define milestones by **objective outcomes, not time periods**; list **out-of-scope items explicitly** in the description (Q3).
- **Never begin out-of-scope work until the new milestone is funded in escrow** (Q3).
- Formal scope/rate/milestone changes go through Upwork's **"Request Changes to an Offer"** feature (Q3).
- Fixed-price has a **14-day auto-approval** window on a submitted milestone if the client is silent (referenced in Q1 sourcing) — use it in delivery follow-ups, don't threaten it.

## Reviews / feedback
- **Client ends the contract → triggers the feedback prompt.** Don't ask for a star number; ask for an honest review, offer to leave yours (Q5).
- Positive private + public feedback is driven by exceeding expectations, early delivery, proactive comms, extra value (Q5).

## Retainers / repeat
- Continue a relationship via **"Propose New Contract"** — keeps existing fee-tier progress with that client (Q6).

## Money + compliance
- Upwork service fee ~**12-13%** since May 2025 → quote take-home ÷ 0.87; never quote take-home directly (Q2).
- **Keep all discussion on-platform** until a contract is active — payment protection + record. Never agree to off-platform payment (Q7).
- Pushing off-platform early (WhatsApp/Telegram), free-sample/test requests, refusing written terms, or starting before the contract is active are **red flags**, not normal asks (Q7).

## Follow-up ceilings
- Pre-hire: **one** follow-up, only with something new; **3+ is spam-flagged** (Q1).
- Post-delivery: gentle check at 3-4 days, an auto-approval reminder at 7-10 days, one "where do things stand" at 2+ weeks (Q1 sourcing).
