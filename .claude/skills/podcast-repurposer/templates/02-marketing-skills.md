# Template 02 — marketing-skills (tactic chain)

**Angle:** Run the transcript through a marketing tactic chain so every piece has an explicit
*job*. Where Template 01 optimizes for craft, this one optimizes for **strategic intent**:
each segment is mapped to a content pillar and a psychological trigger tied to the ICP agent's
fears and desires, and each asset states what it's trying to make the reader feel or do.

**Source methods:** `.claude/skills/marketing-skills/` router — chain
`customer-research → marketing-psychology → content-strategy → copywriting → social`.

Read `references/output-spec.md` for the shape and the client's main voice file for limits and
the ICP. This template defines the *strategic filling*.

---

## Method

### 1. customer-research — anchor on the ICP's fears & desires
Before selecting, list the ICP agent's top fears (commission bleed, feeling unseen, no broker
backing, income uncertainty) and desires (autonomy + real support, recognition, runway to
grow). Selection and copy both serve these. Keep this list at the top of your working notes.

### 2. content-strategy — segment selection by pillar coverage
Pick 3-5 segments so they **cover different pillars** (Feel Secure / Grow with Ease / Belong
Fully) rather than three takes on one theme. A balanced set recruits a wider slice of the ICP.
Each segment names its pillar and the specific fear/desire it speaks to.

### 3. marketing-psychology — assign a trigger per segment
Tag each segment with the psychological lever it pulls and write to it:
- **Loss aversion** (the cost of staying where they are / quitting unprepared)
- **Belonging / in-group** (the culture, the Mama-Bear backing)
- **Autonomy** (be your own boss, with a net)
- **Social proof / authority** (Brenda's track record, agents who made it)
- **Identity** (who you become as an agent here)

### 4. copywriting — every asset has a stated job
For each hook/caption/post, the job is implicit in the writing but explicit in your selection
rationale. Hooks interrupt; captions build the case; long-form posts move the reader toward
"this is the broker who'd back me." The 3 captions = 3 *different jobs* (e.g. agitate the pain
/ paint the desired state / direct invitation), which is what makes them a real A/B.

### 5. social — platform fit, on-brand voice
Apply the voice file's tone and absence signals. Keep platform conventions (LinkedIn = POV /
thought leadership; Facebook = warmer, community; Instagram caption = punchy, scannable).

**Note in your file header** which trigger each segment used — that's this template's
signature and what the comparison will judge it on.
