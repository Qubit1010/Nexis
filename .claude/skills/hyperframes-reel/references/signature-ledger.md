# Signature Ledger

The one rule that makes every reel feel bespoke: **each reel gets its own signature motif, and no two
reels repeat it.** The genome (charcoal canvas, tech-blue voltage, QuicheSans/Urbanist, motion
doctrine) is constant; the *signature* — the central visual device and the world it lives in — is
invented fresh from each post and must be structurally distinct from everything already built.

**Before you design a new reel:** read this ledger. Pick a signature that is a different *kind of
object/world* from every row below — not a restyle of an existing one (a stack, an orb, a graph, a
panel, a chip, a heap, and a document are all different worlds; two "hub-and-spoke graphs" are the
same world).

**After you ship a reel:** append a row here. This file is the memory that keeps the batch diverse.

## How to choose a fresh signature (method)

1. **Mine the post for its native object.** What is the post literally *about*? An OS → a badge/layers.
   A privacy shift → a perimeter/chip. A screen-clicking-vs-API argument → a breaking UI vs a connector
   graph. A text-file optimizer → a document + a gate. The best signature is the post's own subject
   rendered as a physical thing that moves.
2. **Name the recurring texture.** One small repeated element that carries the argument across beats —
   a port+plug (automation), a bolt vs an embedded trace (workspace), a diff line + a check (skillopt),
   an open padlock (local-ai). The texture is what makes the reel read as one idea, not six slides.
3. **Check it against this ledger.** If the object/world matches a row above, pick a different one.
4. **Decide the spine and the cut count.** 0 hard cuts is the default (continuous morph/reveal); use
   the single allowed hard cut only when the story has a real "sever/break" beat (local-ai, ai-os).
5. **Write it into `STORYBOARD.md`** with an explicit "this is deliberately different from [prior
   reels]" paragraph, so the distinctness is a stated design constraint, not an accident.

The genome never changes; the signature never repeats. That combination is the whole look.
