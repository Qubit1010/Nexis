# Beginner - Section 2: The Claude Universe: Every Way to Use It

*Six doors into the same intelligence, and how to know which is yours.*

**Bottom line:** There is one Claude intelligence, reached through six different doors: the web chat, the Desktop app, mobile, Cowork, Claude Code, and the API. You do not need all of them. This chapter is the map so the next ones make sense.

---

## The six doors at a glance

| The door | In plain words | Best for |
|----------|---------------|----------|
| Claude.ai (chat) | Claude in your web browser at claude.ai | Writing, analysis, brainstorming, quick tools |
| Desktop app | The same Claude as a Mac or Windows app | Daily use, plus one home for Cowork and Code |
| Mobile | The phone app, and a remote for your desktop | Quick questions and approving tasks on the go |
| Cowork | An agent that works on your desktop for you | Hands-off file, spreadsheet, and deck work, no code |
| Claude Code | Claude inside a terminal or code editor | Building and fixing software across a project |
| API | The developer plumbing behind Claude | Custom apps and automations that run unattended |

## The six doors, explained

### 1. Claude.ai (web chat) - the front door

A website where you chat with Claude in your browser. Two features are worth knowing now. **Projects** are saved workspaces that remember your context and files, so you are not starting from zero every time. **Artifacts** appear when Claude builds something interactive, like a document, a table, or a small app, and opens it in a side panel you can use and edit. This is where almost everyone starts, and where most of this Beginner level lives.

### 2. Claude Desktop app - the daily home

The same Claude as a proper app on your Mac or Windows machine instead of a browser tab. It is lighter on your computer than a browser, adds a quick pop-up so you can reach Claude from anywhere, and brings the more powerful modes (Cowork and Code) together in one place. If you use Claude every day, install this.

### 3. Claude mobile - the companion

The phone app. Good for quick questions on the go, and it doubles as a remote control: you can check on and approve longer tasks that are running on your computer from your phone. Think of it as the companion, not the main workspace.

### 4. Claude Cowork - the assistant that does the work

Claude as an autonomous assistant that works on your actual desktop. You point it at a folder and give it a goal in plain English, such as "clean up these files," "build this spreadsheet," or "turn this into a slide deck," and it does the work, asking permission as it goes. It is built for non-technical people, with no code or terminal. Covered fully in the Intermediate level.

### 5. Claude Code - the builder

Claude inside a developer's environment, a terminal or code editor, that can build and fix software across many files on its own. It sounds technical, and it is the most powerful door, but you do not have to be a programmer to start. It is the heart of the Advanced level.

### 6. Claude API - the plumbing

Not an app at all. It is what developers use to build Claude into their own products and to run automations that work unattended, with no person watching. Covered near the end of this guide.

## Which door is for me?

Two questions to guide the choice:

- **Who is doing the work?** If it is you, thinking and drafting with Claude helping, use the chat (web or Desktop). If you want Claude to go do something on your computer, use Cowork. If it is building software, that is Claude Code. If it is your own app or an automation that runs by itself, that is the API.
- **How much should it act on its own?** Low, where you steer every step, points to chat. High, where it runs and you review, points to Cowork, Code, or the API.

> **Where to start:** Most readers should live in Claude.ai chat and the Desktop app first; they cover the large majority of everyday needs. Add Cowork when you want work done for you, and Claude Code when you are ready to build.

## Key takeaways

- Same intelligence, six doors. Pick by who does the work and how much autonomy you want.
- Start in Claude.ai chat and the Desktop app; they handle most beginner needs.
- Cowork (Intermediate) does desktop work for you, Claude Code (Advanced) builds software, and the API powers unattended automations.
- You can mix doors. A task often starts in chat and graduates to Cowork or Code as you scale it up.
